Результат:

Исправленная логика шифрования

Исходный байт
01101101
Разбиваем на два полубайта:
Старшие 4 бита: r=6 (0110)
Младшие 4 бита: c=13 (1101)
→
Первый S-бокс (статический)
Берем значение по координатам (r, c)
S1[6][13] = 174
10101110
→
Второй S-бокс (на основе ключа)
Используем значение для определения координат
Новые координаты: (10, 14)
→
Зашифрованный байт
Формируем из координат: (10 << 4) | 14
10101110
174 (0xAE)

Ключевые улучшения:

  • Статический S-Box теперь является биективной перестановкой (все значения уникальны)
  • Реализованы корректные обратные преобразования для дешифрования
  • Улучшена генерация S-Box на основе ключа
  • Гарантирована точная обратимость: decrypt(encrypt(text)) == text
