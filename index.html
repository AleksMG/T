Исходный байт

Разбиваем на два полубайта:
Старшие 4 бита: r=6 (0110)
Младшие 4 бита: c=13 (1101)
→
Первый S-бокс (ключезависимый)
Берем значение по координатам (r, c)
S1[6][13] = 174
10101110
→
Второй S-бокс (ключезависимый)
Используем значение для определения координат
Новые координаты: (10, 14)
→
Третий S-бокс (омофонический)
Случайный выбор из 16 вариантов
S3[10][14] = "Xy"
Результат: 2 символа Base64

Ключевые улучшения:

  • Все S-Box'ы генерируются на основе ключа через SHA-512
  • Омофонический S-Box содержит все 4096 уникальных биграмм Base64
  • Биграммы полностью перемешиваются алгоритмом Фишера-Йейтса
  • Гарантирована точная обратимость: decrypt(encrypt(text)) == text
